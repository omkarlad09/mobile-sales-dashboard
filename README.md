📊 Mobile Sales Dashboard (Power BI)

🔹 Project Overview

This project analyzes mobile sales data using Power BI.
It helps in understanding sales trends, top-performing products, and revenue insights.

🔹 Tools Used

* Power BI
* Excel / CSV

🔹 Key Insights

* 📈 Monthly sales trend analysis
* 📊 Top selling mobile brands
* 💰 Revenue distribution
* 🔎 Region-wise performance



🔹 How to Use

1. Download the `.pbix` file
2. Open in Power BI Desktop

🔹 Project Files

* Mobile_Sales.pbix
* Dataset (CSV/Excel)
